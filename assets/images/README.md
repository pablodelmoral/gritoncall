# Image Assets

Please save the following images in this directory:

1. **bg.png** - The dark wavy background pattern (Image 3)
2. **gockins.png** - The Gockins avatar with red border (Image 2)
3. **logo.png** - The GRIT ON CALL logo (Image 1)

## Image Specifications:

- **bg.png**: Dark background with wavy pattern, will be used as ImageBackground
- **gockins.png**: Circular avatar image (48x48 recommended, will be displayed with red border)
- **logo.png**: GRIT ON CALL text logo in red

## Usage:

- Background: Used in home screen and other main screens
- Gockins Avatar: Used in coach sections and chat
- Logo: Can be used in headers, splash screens, etc.
